# 冒泡排序

```c#
		public static int[] BubbleSort(int[] arr,bool reverse)
        {
            for (int i = 0; i < arr.Length; i++)
            {
                for (int j = i + 1; j < arr.Length; j++)
                {
                    if (arr[i] > arr[j])
                    {
                        var temp = arr[i];
                        arr[i] = arr[j];
                        arr[j] = temp;
                    }
                }
            }

            if (reverse)
            {
                //var newArr = arr.Reverse().ToArray();
                return arr.Reverse().ToArray();
            }

            return arr;
        }
```
